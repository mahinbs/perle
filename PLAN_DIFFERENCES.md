# Premium Plan Differences: IQ Pro vs IQ Max

## Overview

SyntraIQ offers two premium subscription tiers to cater to different user needs:

### IQ Pro - ₹399/month
**Target Audience:** Creators and strategists

### IQ Max - ₹899/month  
**Target Audience:** Teams running mission-critical workflows

---

## Feature Comparison

### ✅ Both Plans Include:

1. **All Premium AI Models**
   - Access to all OpenAI models (GPT-5, GPT-4o, GPT-4o Mini, GPT-4 Turbo)
   - Access to all Grok models (Grok 3, Grok 3 Mini, Grok 4, Grok 4 Heavy, Grok 4 Fast, Grok Code Fast 1, Grok Beta)
   - Access to Gemini 2.0 Latest
   - Access to Claude 4.5

2. **Model Selection**
   - Choose from any premium model
   - Auto mode for smart selection

3. **File Uploads**
   - 5 files per search (vs 2 for free users)
   - Support for images, PDF, CSV, Word, TXT, ODT, Excel

4. **Extended Features**
   - Longer conversation history
   - Priority processing

---

## IQ Pro (₹399/mo) - Specific Features

1. **Priority Access**
   - Priority access to all AI models
   - Faster response times

2. **Unlimited Saved Spaces and Prompts**
   - Save unlimited search queries and prompts
   - Organize your work efficiently

3. **Advanced File Analysis**
   - File analysis up to 200 MB per file
   - Enhanced document processing

4. **Extended Conversation History**
   - Longer chat context retention
   - Better context understanding

---

## IQ Max (₹899/mo) - Everything in IQ Pro, Plus:

1. **Highest Limits**
   - Maximum message and upload limits
   - No restrictions on usage volume

2. **Real-time Collaboration**
   - Shared Spaces for team collaboration
   - Real-time editing and sharing
   - Team workspace features

3. **White-glove Onboarding & Priority Support**
   - Dedicated onboarding assistance
   - Priority customer support
   - Faster response times for support queries

4. **Advanced Analytics and Insights**
   - Usage analytics dashboard
   - Performance insights
   - Team usage reports

5. **Custom Integrations**
   - API access for custom integrations
   - Webhook support
   - Custom workflow automation

6. **Dedicated Account Manager**
   - Personal account manager
   - Custom solutions for your needs
   - Regular check-ins and optimization

## Which Plan Should You Choose?

### Choose **IQ Pro** if you:
- Are a creator or strategist
- Need access to all AI models
- Want faster responses and priority access
- Need unlimited saved spaces
- Work individually or in small teams

### Choose **IQ Max** if you:
- Run mission-critical workflows
- Need team collaboration features
- Require custom integrations
- Want dedicated support and account management
- Need advanced analytics and insights
- Have high-volume usage requirements
